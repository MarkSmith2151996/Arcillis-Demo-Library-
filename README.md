# Arcillis Demo Library

Production-ready demo templates for AI automation consulting — CRM pipelines, document automation, and RAG chatbots.

## Demos

- **CRM Pipeline**: Automated lead capture, qualification, and follow-up workflows.
- **PDF Extractor**: Converts invoices and receipts into structured spreadsheet data with confidence scoring.
- **RAG Chatbot**: Provides knowledge base Q&A with source citations and out-of-scope refusal.

These are deployable systems, not mockups.
