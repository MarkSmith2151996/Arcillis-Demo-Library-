# ARC Toolbar

Compact floating MCP client toolbar.

## Setup (macOS)

1. Install Rust: `curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh`
2. `cd arc-toolbar && npm install`
3. Dev mode: `npm run tauri dev`
4. Build: `npm run tauri build`

Built .app at `src-tauri/target/release/bundle/macos/`
